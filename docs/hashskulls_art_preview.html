HashSkulls Preview

Every skull below is drawn as a 16 by 16 grid using only SVG rects. The point is not a finished renderer patch; it is a concrete direction that can be converted into Solidity helpers without IPFS, images, paths, gradients, or external assets.

16x16 rect-only SVG
Direction: make the silhouette less generic by changing cheek shape, brow weight, jaw width, tooth rhythm, asymmetric cracks, and small accessory profiles. These changes are cheap because they reuse rectangles and short runs, instead of adding complex SVG features.
